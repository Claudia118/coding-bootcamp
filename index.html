Full Stack Web Development

We’ll teach you the languages, tools and techniques you’ll be using as a software developer in the real world. This bootcamp focuses on a foundation in the JavaScript stack plus the latest technologies including React, HTML, CSS, Git, PostgreSQL, and software architecture principles.

  • Front-end with HTML, CSS, and React
  • Server-side programming with Node.js and Express
  • Introduction to Modern JavaScript
  • Version control with Git and GitHub
  • Using relational databases with PostgreSQL
  • Application deployment, DevOps, and containerization using Docker
  • Agile project management methodologies
